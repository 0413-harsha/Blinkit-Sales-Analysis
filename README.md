**Blinkit Sales Analysis Dashboard using Power BI**

**📌 Project Overview**

This project presents an interactive Blinkit Sales Analysis Dashboard built using Power BI to analyze sales performance, product categories, outlet characteristics, customer ratings, and inventory distribution across different outlet types and locations.

The dashboard provides actionable insights into sales trends, product demand, outlet performance, and customer satisfaction, helping business stakeholders make data-driven decisions to improve revenue growth and operational efficiency.


**🛠️ Tools & Technologies Used**
Power BI Desktop
Power Query Editor
DAX (Data Analysis Expressions)
Microsoft Excel
Data Modeling
ETL (Extract, Transform, Load)

**📊 Key KPIs**
Total Sales: $1.20M
Average Sales: $141
Total Items Sold: 8,523
Average Rating: 4.0

**🔄 Data Preparation**
Data Cleaning
Handling Missing Values
Data Transformation using Power Query
Data Modeling
DAX Measures & KPIs

**📈 Dashboard Insights**

Sales Performance

Generated $1.20M in total sales.
Average sales per item reached $141.

Outlet Analysis
Supermarket Type 1 contributed the highest sales.
Medium-sized outlets generated the most revenue.
Tier 3 locations outperformed Tier 1 and Tier 2 locations.

Product Analysis

Fruits & Vegetables, Dairy, Snack Foods, and Household products were among the top-selling categories.
Both Low Fat and Regular products contributed significantly to sales.

Customer Satisfaction

Average customer rating remained consistent at 4/5, indicating positive customer experience.


Outlet Growth
Sales increased steadily across newly established outlets, reflecting successful business expansion.

**💡 Business Recommendations**

Expand operations in Tier 2 and Tier 3 markets.
Focus inventory on high-performing product categories.
Increase investment in medium-sized outlets.
Improve visibility for low-performing products.
